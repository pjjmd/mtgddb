The MTG Demographic Database F.A.Q.

What is it?

The Magic the Gathering Demographic Database (MTGDDB) is a community project to tag the collection of over 10k pieces of fantasy art that make up the collectable card game with gender, race, and other demographic tags. The end goal is a database that can answer questions like 'what is the average casting cost of women in MtG, and how has that changed over time?

Why make this?

Because data is awesome? Because MTG (and Wizards of the Coast in general) is making great strides with representation in it's games, and it's interesting to track that? Because you want to settle a bet about the percentage of black characters in MTG who are bald?

How accurate is this?

Not very, but it's mostly just for fun? All contributors will have to take a tutorial, and all answers will be 'tripple keyed'.

Can I collaborate?

Sure! Do you want access to the database? Do you want to add tags for the community to generate? Contact the site author at Peter@JJMacDonald.com and lets talk!

How did you build this?

It's just JS running on a Parse.API backend. The card data was provided by the MTG JSON project, and the card images are hosted by the MTG Image project.

I sometimes run into a 'broken image', is there a way I can reload the page without having to go through the tutorial?

The tutorial is 5 button clicks long, but if you are really in the groove, you can try this page which bypasses it.

Can I get access to the data?

Sure! Send me an email and I will send you a JSON dump. The client API key for parse is also available in the source code, and the entire database is publicly readable (if you feel comfortable using the parse api).
